You need to develop all your code for this lab in the client.go, server.go and common.go files. You may also modify the ../raft/raft.go file.

After you have made your modifications, you can run all the testing code as follows:

    go test -run 4A

If you wish to run individual tests, look for function names starting with `Test`, e.g., `TestBasic4A` in `test_test.go`. Then run the individual test as follows:

    go test -run TestBasic4A

You can also run the following script to estimate your lab grade:

    /cad2/ece419/bin/ece419-lab4-check A
